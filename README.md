充值这件事，顺的时候三分钟到账，不顺的时候能让人抓狂半天。下面把高频问题都列出来，方便你对症下药。

### 问题1：充值不到账

**可能原因**：
- 链上确认还没完成（尤其ERC20网络慢）
- 充值金额低于最低阈值
- 网络选错了，币转到了错链上

**解决办法**：
- 先去区块链浏览器查这笔交易的确认状态
- PokePay官方承诺15分钟左右到账，超过30分钟直接找客服
- 网络选错是最严重的，币可能找不回来，所以转账前一定核对两边网络一致

### 问题2：到账金额比预期少

**可能原因**：
- 扣了1%充值手续费
- 扣了交易所提币的Gas费
- 汇率转换损耗

**解决办法**：这是正常现象，不是平台扣你钱。PokePay充值手续费固定1%，比如你充100 USDT，到账约99美元。Gas费是给矿工的，跟平台无关。综合算下来，PokePay的充值+消费总损耗大约在2%左右，在同类U卡里属于偏低的水平。

### 问题3：KYC认证失败

**可能原因**：
- 填的姓名和证件不一致
- 证件照片模糊或反光
- 人脸识别环境光线差

**解决办法**：重新提交，姓名严格按证件拼，照片在自然光下拍，别戴眼镜和帽子。

### 问题4：开卡时提示余额不足

**原因**：你没充值就想去开卡。PokePay的流程是先充值、后开卡，账户里必须有钱才能扣开卡费。建议第一次充25-30 USDT，覆盖开卡费+手续费绰绰有余。
